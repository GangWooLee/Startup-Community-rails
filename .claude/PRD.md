## 5. 비기능 요구사항

### 5.1 성능
- 페이지 로드: < 2초
- API 응답: < 500ms
- 동시 접속: 1,000명 지원

### 5.2 보안
- HTTPS 필수
- 비밀번호 암호화 (bcrypt)
- CSRF 보호
- SQL Injection 방지
- XSS 방지

### 5.3 확장성
- 수평 확장 가능 구조
- 캐싱 전략
- DB 인덱싱

### 5.4 사용성
- 모바일 반응형 디자인
- 접근성 (WCAG 2.1 AA)
- 다국어 지원 (선택)
