## 1. 제품 개요

### 1.1 비전 (Product Vision)
"아이디어·사람·외주가 한 공간에서 연결되는 최초의 창업 커뮤니티"

### 1.2 문제 정의 (Problem Statement)
한국의 초기 창업자들은 다음과 같은 문제를 겪고 있습니다:

1. **커뮤니티는 있으나 외주 정보와 연결되지 않는다**
2. **크몽/위시켓은 외주만 있고 커뮤니티 성격이 없다**
3. **아이디어에 대한 피드백을 받을 기회가 부족하다**
4. **신뢰 기반 프로필이 없어 사람을 구하기 어렵다**
5. **정보가 파편화되어 있다** (에브리타임/인스타/카카오톡/구글폼)

### 1.3 솔루션
커뮤니티 기반의 신뢰 데이터와 외주 기능을 결합하여
처음 창업하는 사람들이 시행착오 없이 시작할 수 있도록 돕는 통합 플랫폼

### 1.4 MVP 목표
"커뮤니티 활동 → 프로필 → 외주 공고"가 자연스럽게 연결되는 사용자 흐름 구축

**구체적 목표**:
- 사용자가 글을 올리고 대화를 나눌 수 있다
- 글쓴이의 프로필에서 그 사람의 외주 구인/구직 게시물을 확인할 수 있다
- 외주 공고 페이지는 최소 정보만 포함해도 클릭해볼 수 있다
- 마이페이지에서 내 활동(스크랩/프로필 수정) 관리를 할 수 있다

## 2. 타겟 사용자

### 2.1 Community User (커뮤니티 사용자)
- **프로필**: 대학생/초기 창업자
- **Pain Points**:
  - 창업 관련 질문을 올리고 피드백을 받을 곳이 없다
  - 신뢰할 수 있는 정보를 찾기 어렵다
  - 비슷한 관심사를 가진 사람들을 만날 기회가 없다
- **Needs**:
  - 창업 관련 질문과 피드백
  - 마음에 드는 사람의 프로필 확인
  - 누가 어떤 프로젝트를 구하는지 파악

### 2.2 Outsourcing Client (구인자)
- **프로필**: 외주를 맡기고 싶은 소규모 창업자
- **Pain Points**:
  - 프로필 기반으로 사람이 신뢰할 만한지 알기 어렵다
  - 내가 올린 구인 글을 한 곳에서 관리하기 어렵다
- **Needs**:
  - 신뢰 기반 프로필 확인
  - 구인 글 통합 관리

### 2.3 Freelancer/Talent (구직자)
- **프로필**: 사이드프로젝트를 하고 싶은 개발자/디자이너/기획자
- **Pain Points**:
  - 내가 제공할 수 있는 서비스를 정리해 올릴 곳이 없다
  - 포트폴리오를 보여줄 프로필 페이지가 없다
- **Needs**:
  - 서비스 제공 정보 게시
  - 포트폴리오 형태의 프로필 페이지

### 2.4 General User (일반 사용자)
- **프로필**: 창업 관련 인사이트를 얻고 싶은 사용자
- **Needs**:
  - 스크랩해둔 게시물 관리
  - 유용한 정보 저장

## 3. 핵심 기능 (Core Features)

## ✔️ (1) Community 게시판

**기능 목표**:
창업자/개발자/디자이너가 양질의 대화를 나누는 공간을 만들고, 프로필로 연결되는 첫 접점을 제공한다.

**주요 기능**:
- 글쓰기
- 댓글
- 좋아요
- 스크랩
- 공유 (선택)
- 글쓴이 프로필 클릭 기능

**UX Requirements**:
- 게시글 카드에는 반드시 "작성자 프로필 썸네일 + 이름 + 기본 역할(직무)" 포함
- 카드 클릭 시 상세 페이지로 이동
- 프로필 클릭 시 프로필 페이지로 이동

**사용자 스토리**:
```
As a 초기 창업자
I want to 창업 관련 질문을 올리고 피드백을 받고 싶다
So that 시행착오를 줄이고 올바른 방향으로 나아갈 수 있다
```

**Acceptance Criteria**:
- [ ] 로그인한 사용자는 글을 작성할 수 있다
- [ ] 모든 사용자는 게시글 목록을 볼 수 있다
- [ ] 게시글 카드에 작성자 정보가 표시된다
- [ ] 댓글을 작성하고 조회할 수 있다
- [ ] 좋아요와 스크랩 기능이 작동한다

---

## ✔️ (2) Profile 페이지 (구인/구직 통합 프로필)

**기능 목표**:
커뮤니티 활동 ↔ 외주 기능을 연결하는 핵심 페이지.
"그 사람은 어떤 사람이고, 어떤 일을 구하거나 제공하는지" 한눈에 보여주는 구조.

**구성**:
- **프로필 정보**
  1. 프로필 사진
  2. 이름 / 역할 (Founder, Developer 등)
  3. 간단한 소개
- **탭 3개**
  1. Posts – 작성한 커뮤니티 글
  2. Job Posts (구인) – 외주 요청 글
  3. Talent Listings (구직) – 제공 가능한 서비스
- 탭 클릭 시 해당 리스트 노출
- 외주 글 클릭하면 외주 상세 페이지로 이동

**UX Requirements**:
- 카드 디자인은 모두 동일한 규격
- 리스트는 무한스크롤 or 페이지네이션

**사용자 스토리**:
```
As a 외주 구인자
I want to 프로필을 기반으로 사람이 신뢰할 만한지 알고 싶다
So that 믿을 수 있는 사람에게 프로젝트를 맡길 수 있다
```

**Acceptance Criteria**:
- [ ] 프로필 페이지에 사용자 기본 정보가 표시된다
- [ ] 3개의 탭(Posts, Job Posts, Talent Listings)이 작동한다
- [ ] 각 탭에서 관련 콘텐츠를 볼 수 있다
- [ ] 프로필 소유자는 정보를 수정할 수 있다

---

## ✔️ (3) Outsourcing 공고 페이지 (구인/구직)

**기능 목표**:
최소 기능으로 외주 정보 탐색을 가능하게 한다.

**포함 요소**:
- 제목
- 카테고리 (개발/디자인/PM/마케팅)
- 프로젝트 타입 (단기/장기/일회성)
- 설명
- 예산 (optional)
- 작성자 프로필
- 지원(문의) 버튼 (1차 MVP에서는 기능 없이 UI만 존재 가능)

**UX Requirements**:
- 상단에 작성자 미니 프로필 표시
- 버튼 위치는 항상 우측 or 하단 고정
- 복잡한 기능 없음

**사용자 스토리**:
```
As a 프리랜서/구직자
I want to 내가 제공할 수 있는 서비스를 정리해 올리고 싶다
So that 프로젝트를 찾는 사람들이 나를 발견할 수 있다
```

**Acceptance Criteria**:
- [ ] 구인 공고를 생성할 수 있다
- [ ] 구직 정보를 생성할 수 있다
- [ ] 공고 목록을 카테고리별로 필터링할 수 있다
- [ ] 공고 상세 페이지에서 작성자 프로필로 이동할 수 있다

---

## ✔️ (4) My Page (마이페이지)

**기능 목표**:
프로필 관리와 내가 저장한 정보 접근 제공.

**구성**:
- 프로필 수정
- 내가 스크랩한 글
- 내가 올린 구인·구직 글
- 로그아웃

**사용자 스토리**:
```
As a 일반 사용자
I want to 내가 스크랩해둔 게시물을 한 번에 보고 싶다
So that 나중에 다시 찾아볼 수 있다
```

**Acceptance Criteria**:
- [ ] 프로필 정보를 수정할 수 있다
- [ ] 스크랩한 글 목록을 볼 수 있다
- [ ] 내가 작성한 구인/구직 글을 볼 수 있다
- [ ] 로그아웃 기능이 작동한다

## 6. 비기능 요구사항 (Non-functional Requirements)

### 6.1 성능
- 초기 로딩 속도 최소화 (간단한 DB 구조)
- 페이지 로드: < 2초

### 6.2 보안
- HTTPS 필수
- 비밀번호 암호화 (bcrypt)
- CSRF 보호
- SQL Injection 방지
- XSS 방지

### 6.3 사용성
- **반응형 디자인 (모바일 우선)**
- 초기엔 로그인 없이도 내용 볼 수 있도록 허용 가능
- 회원가입은 최소정보(이름·역할·학과 또는 직무·한줄소개)

### 6.4 확장성
- 수평 확장 가능 구조
- DB 인덱싱
- 캐싱 전략

## 8. 로드맵

### Phase 1: MVP (Week 1-4)
**목표**: 커뮤니티 + 프로필 + 외주 기본 흐름 구축

- Week 1: 프로젝트 셋업 & 인증
- Week 2: 커뮤니티 기능
- Week 3: 프로필 & 외주 기능
- Week 4: 마이페이지 & 통합 테스트

### Phase 2: Enhancement (향후)
- AI 기반 프로젝트 매칭 알고리즘
- 커뮤니티 사용자 평판/신뢰도 점수
- 동아리/학회/프로젝트 팀 정보 DB화
- 공동창업자 추천 기능
- IR 콘텐츠/멘토링 기능
- 외주 결제/정산 기능
- DM/채팅 기능
- 액티비티 기반 AI 개인 피드
